== Description ==

HTML代码优化工具（Clear HTML Tags）是一款站长实用的WordPress文章编辑辅助插件，可以帮助站长快速实现删除HTML代码不需要的常见HTML标签及标签属性，常用的代码格式优化。

### 1.支持删除超过40种的代码标签且支持自定义设置。

支持删除的HTML代码标签包括：
* **常见标签**-a/div/span/article/figcaption/figure/br/section/blockquote/em/b/i/strong/h1/h2/h3/h4/h5/h6/p/img/video。
* **表格标签**-table/tbody/thead/tfoot/td/th/tr/col/caption/colgroup。
* **列表标签**-ul/li/ol/dl/dt/dd。
* **其他标签**-detail/header/footer/hr/iframe/ins/body/head/html/ruby/title/script/nav。

### 2.支持删除超过10种的代码标签属性且支持自定义设置。

支持删除的常见HTML代码标签属性包括：id/class/style/srcset/sizes/width/height/data-*/rel/alt等.

### 3.支持代码格式优化。

支持段落开头空格删除，段落之间自动空行，图像之间自动空行，图像与段落间自动空行，H标题与段落间自动空行。

全新的版本在用户体验上进行了优化，由原来的文章编辑操作时设置插件参数改为插件设置界面设置默认参数，文章编辑时可临时调整当前操作参数，提升插件的工作效率。

== Installation ==

方式1：在线安装(推荐)

1. 进入WordPress仪表盘，点击“插件-安装插件”：
* 关键词搜索“HTML代码优化工具”，找搜索结果中找到“HTML代码优化工具”插件，点击“现在安装”；
* 或者点击“上传插件”-选择“HTML代码优化工具”插件压缩包clear-html-tags.zip，点击“现在安装”。
2. 安装完毕后，启用 `HTML代码优化工具` 插件.
3. 通过“设置”->“HTML代码优化工具” 进入插件设置界面进行插件参数设置.
4. 编辑文章时，点击编辑器“清楚代码”按钮执行操作即可。

方式2：上传安装

FTP上传安装
1. 解压插件压缩包clear-html-tags.zip，将解压获得文件夹上传至wordpress安装目录下的 `/wp-content/plugins/`目录.
2. 访问WordPress仪表盘，进入“插件”-“已安装插件”，在插件列表中找到“HTML代码优化工具”，点击“启用”.
3. 通过“设置”->“HTML代码优化工具” 进入插件设置界面.
4. 编辑文章时，点击编辑器“清楚代码”按钮执行操作即可。

仪表盘上传安装
1. 进入WordPress仪表盘，点击“插件-安装插件”；
2. 点击界面左上方的“上传按钮”，选择本地提前下载好的插件压缩包clear-html-tags.zip，点击“现在安装”；
3. 安装完毕后，启用"HTML代码优化工具"插件；
4. 通过“设置”->"HTML代码优化工具"进入插件设置界面.
4. 编辑文章时，点击编辑器“清楚代码”按钮执行操作即可。


== Frequently Asked Questions ==

= 为什么该插件在古腾堡编辑器不生效？ = 
插件暂时不支持古腾堡编辑器，我们将来会根据古腾堡编辑器的发展情况及成熟程度再考虑兼容。关闭古腾堡编辑器使用经典编辑器，可以通过安装插件Disable Gutenberg实现。

= 在升级插件版本时遭遇429 too many requests报错？ = 
这是由于WordPress官方服务器限制了中国区域请求导致，你可以通过安装<a href="https://www.wbolt.com/plugins/kill-429?utm_source=wp&utm_medium=link&utm_campaign=cht" rel="friend" title="Kill 429">Kill 429</a>这个插件来解决这个问题。

= 自定义代码标签格式是否需要输入完整的<标签>和</标签>？ = 
不需要，你无需输入<>及</>，只需要输入标签名即可。
