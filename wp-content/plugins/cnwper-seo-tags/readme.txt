== Description ==

WordPress自动内链关键字插件（CNWPer SEO Tags）,能够自动化帮助站长随机或者严格匹配当前网站文章相关关键字插入到TAGS中，且实现自动内链。

## 插件特点

1. 自动实现全站TAGS内链功能开启
2. 批量给指定范围文章插入TAGS关键字
3. 预设相关关键字 编辑文章时自动加入文章TAGS中
4. 设置TAGS最高值，减少搜索引擎判断作弊风险
5. 可选严格匹配标题和内容 做到关键字相关性

== Installation ==

* 把插件文件夹上传到/wp-content/plugins/目录下<br />
* 在后台插件列表中激活插件<br />
* 在《自动关键字设置》菜单设置自动关键字选项和批量执行任务<br />


== Frequently Asked Questions ==

* 1.当发现插件出错时，开启调试获取错误信息。
* 2.我们可以选择备份对象存储或者本地同时备份。
* 3.批量插入关键字的时候，注意数量和相关性匹配，过多插入关键字可能会误判作弊
* 4.预设的关键字建议频率数值不要太高，单篇文章关键字数控制在3-5个
